Jitsi Meet on Docker for Synology

Jitsi is a set of Open Source projects that allows you to easily build and deploy secure video conferencing solutions. Jitsi Meet is a fully encrypted, 100% Open Source video conferencing solution that you can use all day, every day, for free with no account needed. This package is based on a repository that contains the necessary tools to run a Jitsi Meet stack on Docker using Docker Compose wrapped in scripts and UI.

Quick Start

Basics had been set during install. Post changing configuration run jitsi-cli reset to apply it. Run jitsi-cli refresh weekly for new releases. Using a NAT router ensure to set DOCKER_HOST_ADDRESS to external ip route streaming port 10000(udp). Use Synology Reverse-Proxy to route https-443 to container http-8080.

Architecture

A Jitsi Meet installation can be broken down into the following components:
  • a web interface (for landing) & a XMPP server (for user authentication)
  • a conference focus component & a video router (could be more than one)
  • a SIP gateway for audio calls & a Broadcasting Infrastructure for recording or streaming a conference.
